Website

【自主制作】架空ヘアケアブランドサイト

URL https://yoshino1006.github.io/feuille/
概要

架空ヘアケアブランドサイトのデザインカンプ作成、コー ディングを行いました。

クライアント情報

ブランド名：feuille (葉)
ブランドコンセプト：ライフステージの変化が多い30代の 女性に寄り添った、植物オイルであるアルガンオイル、ホホ バ油を取り入れたヘアケアシリーズ。

要件定義

目的：ブランドの知名度向上。
ターゲット：ライフステージ(仕事や人間関係、妊娠など)の
変化が多い30代の女性
トンマナ：・年代に合わせた上品なデザイン。 　　　　　・ナチュラルさが欲しい。

ペルソナ

33歳/さきこさん/女性/雑貨店で働く正社員
自然派を好む。Instagramを見るのが習慣である。
31歳のときに妊娠。また産休・育休が終わって、仕事復帰 したが、休む前よりもストレスが溜まりやすくなり、体質 が変わっているのを実感した。最近、シャンプーの銘柄を 変えようと思っており、自分の体質に合ったヘアケアブラ ンドを検討している。

制作期間

1ヶ月

使用ツール

Photoshop / Illustrator / XD / HTML / CSS / jQuery

制作

2022年

工夫した点

・ブランド名であるfeuille （葉）からパッケージデザインを 考えました。木のように長く使い続けられるヘアブランドに なって欲しいという意味が込められています。ターゲットに 合わせて、上品かつシンプルなデザインに仕上げました。
・緑のグラデーションをサイトの背景と商品の背景に敷く ことで、ナチュラルなイメージと上品さを兼ね備えたデザイ ンに仕上げました。
・ヘッダーは表示させず、ハンバーガーメニューにすること で、FVの印象を強くしました。
